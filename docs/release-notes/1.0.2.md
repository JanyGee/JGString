# JGString 1.0.2

JGString 1.0.2 is a maintenance release focused on reliability, discoverability, and easier adoption.

## Highlights

- Added Swift Package Manager support
- Added XCTest coverage for the public string helpers
- Fixed edge cases in `between`, `join`, `decodeHTML`, trimming helpers, and integer subscripts
- Replaced Travis CI with GitHub Actions
- Reworked README and CocoaPods metadata for better search visibility

## Good For

- Swift string formatting
- slug generation
- camelCase conversion
- HTML entity decoding
- trimming and whitespace cleanup
- number, boolean, and date parsing

## Suggested Release Title

`JGString 1.0.2: SPM support, test coverage, bug fixes, and better docs`
