# Changelog

All notable changes to this project will be documented in this file.

## 1.0.2 - 2026-04-03

- Added Swift Package Manager support with `Package.swift`
- Added XCTest coverage for the public `String` extension API
- Fixed edge cases in `between`, `join`, `decodeHTML`, `trimmedLeft`, `trimmedRight`, `times`, and integer range subscripts
- Improved helper behavior for whitespace handling, initials extraction, alphanumeric checks, and empty substring counting
- Replaced Travis CI with GitHub Actions
- Refreshed README and CocoaPods metadata to better describe the library's capabilities
